### 介绍
python appium-desktop UI自动化测试

### 环境
* Python3
* Android SDK
* JDK1.8
* unittest
* appium client    
* appium Server    



### 主要内容

* PO模式
* 每条用例运行完成自动截图
* 生成html测试报告



### 运行

* 单个测试文件运行
~~~
python3 *_case.py //case目录下
~~~

* 批量运行
~~~
python3 run_test.py //case目录下
~~~
